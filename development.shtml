freenode's current generation ircd project, ircd-seven, is a branch of the charybdis ircd which has replaced the legacy hyperion code base which was used previously. ircd-seven is currently being developed upon and is continually being improved.

The source code, as well as a wiki and bug tracker are available in Redmine. If you're interested in helping with, or just following, the development of ircd-seven, join us in #freenode-seven or on the ircd-seven test network.

There is a public test network for ircd-seven running on testnet.freenode.net port 9002, or 9003 for SSL connections. This is independent of the main freenode network, so your normal channels may not exist -- if you want to test the new server software, come and join us in #freenode there.

Thanks in advance for any help you can provide!

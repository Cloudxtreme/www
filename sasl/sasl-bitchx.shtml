Information about using SASL with BitchX is in the Readme file by Dan Marshal. Search for "SASL"; it's about halfway down the page.

  1. Edit or create a file named .ircservers in your home directory.

  2. Add or edit the line for freenode to look like:

    chat.freenode.net:::NickName::SASL-username:NickServ-password

    Replace NickName with the nick you want to use, SASL-username with your primary nickname, and NickServ-password with your services password. The next time BitchX is started, it will connect and identify automatically using SASL.
